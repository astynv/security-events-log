# React + TypeScript + Vite

# Структура

Сервисы поделены на страницы (pages), если в дальйнешем будет больше модулей, чем просто таблица событий, для каждого из них свои компоненты, функции и стили, и общие для всего проекта

## Локальный запуск

Необходимо иметь node, установленный на ПК, версией выше 18 (здесь работает v20.19.0)

Для запуска:

```bash
npm install
```

```bash
npm run dev
```
